# Thailand & Hong Kong Master Trip Plan

A static GitHub Pages itinerary site.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Add these files to the repo:
   - `index.html`
   - `style.css`
   - `script.js`
3. Commit and push.
4. Go to **Settings → Pages**.
5. Source: **Deploy from branch**.
6. Branch: `main`, folder: `/root`.
7. Open the published GitHub Pages URL.

## Local preview

Open `index.html` in your browser.
